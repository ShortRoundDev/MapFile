Detailed Description

Contains all the functions and error codes for parsing text and lexemes into MapFile structs.

int MF_CountAttributes ( struct MF_Lexeme *  lexemes)

Counts the number of Key Value Pairs at the given hierarchy of the lexeme list. Halts when it encounters to first opening bracket {, or end of the list.

Parameters
[In]lexemes The lexeme to begin counting at
Returns
The number of attributes in the entity. 0 is bad (all entities need at least a classname).

◆ MF_Parse()

BOOL MF_Parse ( char *  text,
struct MF_Map *  map 
)

Parses the text of a mapfile into its relevant MF_Map structs. This runs allocations every step of the way. If there is an error in parsing, this returns false and an error code is set. The developer can get error information with MF_GetErrCode() and MF_GetErrMessage()

Parameters
[In]text The text of the mapfile
[Out]map A pointer to where the map data will be stored after parsing
Returns
True on success, False on failure
