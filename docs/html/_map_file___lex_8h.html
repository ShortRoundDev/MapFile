PRIVATE size_t MF_LexWhitespacePaddedString ( char *  string)

Returns the length of the lexeme ending with whitespace or a null terminator. All characters are matched up until whitespace, a newline, or the null terminator, so this function should never return 0 (unless the input itself starts with whitespace, which should never happen in the core MF_Lex loop).
